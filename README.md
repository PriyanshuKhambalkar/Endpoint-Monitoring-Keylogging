## рҹ“– What is This Project?

**Endpoint Monitoring вҖ” Keylogging** is an educational Python project that demonstrates how keystroke capture works at the system level. It is designed to help cybersecurity students and developers understand the mechanics of input monitoring вҖ” a technique frequently studied in endpoint security, digital forensics, and ethical hacking courses.

By understanding how keyloggers work, security professionals can better design detection systems, write better endpoint protection software, and identify malicious activity on compromised systems. This project makes that learning accessible through clean, readable Python code.

---

## вңЁ Features

- вҢЁпёҸ **Keystroke Capture** вҖ” Records all keyboard input in real time
- рҹ“„ **File Logging** вҖ” Saves captured keystrokes to a local log file
- рҹ”‘ **Special Key Handling** вҖ” Detects and logs special keys (Enter, Space, Backspace, etc.)
- рҹ•җ **Timestamp Logging** вҖ” Each session logged with date and time
- рҹӣ‘ **Clean Stop** вҖ” Graceful exit on defined key or keyboard interrupt
- рҹӘ¶ **Lightweight** вҖ” Single script, minimal dependencies

## рҹ§  How It Works

```
Program starts
      в”Ӯ
      в–ј
pynput listener attached to keyboard
      в”Ӯ
      в–ј
Key pressed by user
      в”Ӯ
      в”ңв”Җв”Җ Regular key  вҶ’  Character appended to log
      в”ңв”Җв”Җ Special key  вҶ’  [KEY_NAME] appended to log
      в””в”Җв”Җ Stop key     вҶ’  Listener stopped, log saved
      в”Ӯ
      в–ј
Keystrokes written to log file
      в”Ӯ
      в–ј
Session ends вҖ” log file saved locally
```

The `pynput` library attaches a non-blocking listener to the operating system's keyboard input stream. Every key event is intercepted, classified as either a regular character or a special key, formatted into a readable string, and appended to the output log file. The listener runs in a background thread, keeping the script responsive while capturing all input.

## в–¶пёҸ Usage

```bash

cd Endpoint-Monitoring-Keylogging
```

**2. Install dependencies:**
```bash
pip install -r requirements.txt
```

**3. Run the keylogger:**
```bash
python keylogger.py
```

**4. Stop the logger:**
```
Press Ctrl+C  or the defined stop key
```

**5. View the log:**
```bash
cat keylog.txt        # Linux / macOS

```
