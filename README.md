# 🖥️ Endpoint Monitoring – Keylogging

Endpoint Monitoring – Keylogging is an educational project demonstrating how keystrokes can be captured in a controlled environment. The tool logs keyboard input and provides insight into how keylogging techniques work, useful for learning about system monitoring and input capture.

---

## 🚀 Features

- Captures keystrokes from the keyboard  
- Logs input into a file  
- Easy to run and understand  
- Educational and informative

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries Used:** `pynput` (for capturing keyboard input)

---

## 📁 Project Structure

```text
Endpoint-Monitoring-Keylogging/
├── keylogger.py     # Main keylogging script
├── README.md        # This documentation
└── requirements.txt # Required Python libraries
```

---

## ⚙️ Requirements

Before running this project, ensure you have:

- Python 3 installed  
- Required dependencies installed via pip

Install dependencies using:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

1. Open a terminal in the project directory.

2. Run the keylogger script:

```bash
python keylogger.py
```

3. The script will start logging keyboard input and save it into a file.
